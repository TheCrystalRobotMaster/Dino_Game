# Dino_Game is based on the STM32F103C8T6 microcontroller
### This mini-game was developed using a development board designed by the Bilibili uploader "keysking". The schematic diagram for the development board is located in the "Schematic_diagram" folder. Please review it before modifying the source code to identify the corresponding GPIO pins

To download this mini-game, you need to use the STM32CubeIDE software; the game itself is developed using the HAL library.
Usage instructions: Clone the repository files to your local machine, open the file with the .ioc extension using CubeIDE, and then flash it within the IDE.
